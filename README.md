# telegramBot

aliyun oss文件小助手

## prerequisite

```
pip install expiringdict
pip install pyTelegramBotAPI
```

```
mv config_example.py config.py
update TOKEN to you telegrambot API token
nohup python main.py
```

## 支持功能

- 上传单张图片并返回网络地址
